## 📌 How to Use  
1. **Install Dependencies**  
   ```bash
   pip install -r requirements.txt
   ```  
2. **Run the Script**  
   ```bash
   python remove_watermark.py
   ```  
3. **Select Image** – Upload your AI-generated image  
4. **Process & Save** – Get your watermark-free image
